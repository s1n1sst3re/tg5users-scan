# Проверка доступности имени пользователя на Fragment

Этот Python-скрипт проверяет доступность 5-символьных имен пользователей на Fragment.com (платформе для аукционов имен пользователей Telegram).

## Возможности

- Генерирует все возможные 5-символьные имена пользователей, используя строчные буквы (a-z) и цифры (0-9)
- Проверяет доступность каждого имени пользователя на Fragment.com
- Определяет, является ли имя пользователя:
  - Доступным (перенаправляет на страницу поиска, что означает, что оно не существует)
  - Предметом аукциона (имеет страницу с информацией об аукционе)
  - Занятым (в остальных случаях)

## Как это работает

Скрипт работает следующим образом:

1. Выполняет HTTP-запросы к `https://fragment.com/username/{username}`
2. Проверяет URL-адрес ответа и содержимое:
   - Если URL-адрес изменяется на `https://fragment.com/?query={username}`, имя пользователя не существует и доступно
   - Если URL-адрес остается `https://fragment.com/username/{username}` и содержит термины, связанные с аукционом, оно участвует в аукционе
   - В противном случае, оно считается занятым

## Использование

```bash
python3 fragment_username_checker.py
```

Скрипт:
- Сгенерирует и проверит 5-символьные имена пользователей (a-z, 0-9)
- Покажет статус доступности для каждого имени пользователя
- По умолчанию ограничивается 10 именами пользователей для демонстрации

## Важные замечания

- Существует 36^5 = 60 466 176 возможных 5-символьных имен пользователей
- Проверка всех имен пользователей займет очень много времени
- Уважайте серверы Fragment.com, не делая слишком много запросов слишком быстро
- Скрипт включает задержку 0,5 секунды между запросами
